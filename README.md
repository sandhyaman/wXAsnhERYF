# Python图书信息管理系统 python844

## 项目概述

本项目是基于Python语言的图书信息管理系统，使用Django框架进行开发，实现了图书信息的增删改查等管理功能，界面采用HTML、CSS进行设计，并通过JavaScript进行交互。

## 技术栈

- 后端：Python + Django
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

### 基础功能

- 登录注册
- 分页查询

### 管理功能

- 书籍信息管理
 - 书籍排序
 - 书籍搜索
- 出版社信息管理
- 作者信息管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python环境
- Django框架
- MySQL数据库
- 支持HTML、CSS和JavaScript的Web浏览器

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 配置MySQL数据库
4. 部署前端静态文件
5. 运行Django项目
